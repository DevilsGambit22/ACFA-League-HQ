# ACFA League HQ Sidebar v1.1.0

Refined specifically for use as a Chess.com club sidebar/dashboard.

## Updates
- Removed the redundant Open League HQ button.
- League Forums links to the League HQ forum area.
- League Operations links to the club events page.
- Removed inactive Private Headquarters, Club Representatives, and Season Records controls.
- Rewrote About to explain the league and its event formats.
- Converted uploaded MP4 music into browser-friendly MP3 audio.
- Added reliable radio controls, progress, duration, playlist, and error handling.
- Album artwork rotates and glows while audio plays.
- Soundwave animates only while music is playing.
- Added an animated Lady Justice hero, rotating light, gold chess matrix, scanner, card reveals, shimmer, glows, and page motion.
- Preserved live Chess.com Public API member avatars, ratings, and Commissioner profile.

## Upload instructions
Replace the repository contents with the contents of this ZIP. At minimum, replace:
- `index.html`
- `assets/music/`
